# 录屏：避免上下文内容特定

推特上的反馈让我们再一次有机会来review一个设计。

这一次， Fran Pérez给我发了一个有部分选择器标注的命名例子，我查看了他的命名规范，给出了一些使得系统更加灵活的建议。

命名一个东西是很困难的，我希望通过这个视频能够让大家知道良好的命名规范能够使得命名更加容易，而不是想出名字都难。
